# LLM One Pager 功能 - 使用指南

## 概述

LLM One Pager 是飞书机器人的自动公司分析功能，使用 AI 的知识库生成结构化的公司研究报告。

**特点：**

- ✅ 无需外部 API，立即可用
- ✅ 支持中国上市公司和知名企业
- ✅ 生成结构化 Markdown 报告
- ✅ 包含业务、财务、亮点、风险分析

## 快速开始

### 1. 启用功能（默认已启用）

在 OpenClaw 配置中添加（可选）：

```yaml
channels:
  feishu:
    llmOnePager:
      enabled: true # 默认为 true
      systemPrompt: "自定义分析师提示词" # 可选
```

或通过环境变量：

```bash
export LLM_ONEPAGER_SYSTEM_PROMPT="你是一位专业的投资分析师..."
```

### 2. 使用方式

#### 私聊（DM）

直接发送公司名称：

```
贵州茅台
中国平安
阿里巴巴
```

或使用显式格式：

```
onepager 腾讯
一页纸 京东
```

#### 群聊

需要 @ 机器人：

```
@机器人 贵州茅台
@机器人 onepager 中国平安
```

## 生成的报告格式

机器人将自动生成包含以下内容的 One Pager：

### 📋 基本信息

- 公司名称
- 股票代码（如适用）
- 所属行业
- 成立时间
- 总部位置

### 💼 业务摘要

核心业务、产品/服务、商业模式、市场地位

### 📊 财务摘要

- 营收规模
- 利润水平
- 增长率
- 关键财务指标

### ✨ 投资亮点

- 竞争优势
- 增长驱动因素
- 市场机会

### ⚠️ 风险提示

- 行业风险
- 公司特定风险
- 监管风险

## 配置选项

### channels.feishu.llmOnePager.enabled

**类型**: `boolean`  
**默认**: `true`

是否启用 LLM One Pager 功能。

```yaml
channels:
  feishu:
    llmOnePager:
      enabled: false # 禁用功能
```

### channels.feishu.llmOnePager.systemPrompt

**类型**: `string`  
**默认**: 内置专业分析师提示词

自定义 AI 生成报告的指导提示词。

```yaml
channels:
  feishu:
    llmOnePager:
      systemPrompt: |
        你是一位专业的价值投资分析师。
        重点关注：
        1. 长期竞争优势
        2. 管理层质量
        3. 财务稳健性
        4. 估值水平
```

## 环境变量

### LLM_ONEPAGER_SYSTEM_PROMPT

设置系统提示词（如果配置文件中未设置）：

```bash
export LLM_ONEPAGER_SYSTEM_PROMPT="自定义提示词"
```

## 注意事项

1. **信息时效性**：报告基于 AI 的知识库，可能不包含最新的财务数据
2. **免责声明**：报告仅供参考，不构成投资建议
3. **公司覆盖**：优先支持知名上市公司，小型或新上市公司可能信息有限
4. **查询限制**：
   - 公司名称长度：2-50 字符
   - 不支持多行或包含 URL 的查询
   - 群聊中必须 @ 机器人

## 故障排查

### 机器人没有响应

1. 检查配置是否启用：

   ```bash
   openclaw config get channels.feishu.llmOnePager.enabled
   ```

2. 确保在群聊中 @ 了机器人

3. 检查查询格式是否符合要求（2-50 字符，无特殊符号）

### 生成的报告信息不足

公司可能不在 AI 的知识库中，AI 会诚实告知。建议：

- 查询更知名的公司
- 访问公司官网或证券交易所获取官方信息

### 想要实时行情数据

如需实时行情，可以：

1. 启用 Tushare One Pager（需要配置 Token）
2. 使用专业财经工具查询

## 技术细节

### 实现原理

1. **消息检测**：检测用户输入是否符合公司查询格式
2. **提示注入**：将系统提示词和用户查询组合
3. **LLM 生成**：由 OpenClaw 的 AI agent 生成结构化报告
4. **格式化输出**：以 Markdown 格式返回飞书

### 扩展性

可以通过 `systemPrompt` 自定义分析风格：

- 价值投资视角
- 成长股分析
- ESG 评估
- 行业对比分析
